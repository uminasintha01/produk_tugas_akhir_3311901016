# cordova-plugin-extended-device-information



## Installation

    cordova plugin add cordova-plugin-extended-device-information

    Browser Console
    cordova.plugins['extended-device-information'].freestorage

## Properties

- device.memory
- device.cpumhz
- device.totalstorage
- device.freestorage
